Simple PHP OOP Student Management System

Used OOP concepts:
1. Class -> Person, Student, StudentRepository
2. Object -> new Student(), new StudentRepository()
3. Property -> id, name, course, phone, file
4. Method -> getId(), getName(), save(), getAll(), toCSV(), fromCSV()
5. Encapsulation -> private/protected properties + getter methods
6. Inheritance -> Student extends Person
7. Abstraction -> abstract Person class
