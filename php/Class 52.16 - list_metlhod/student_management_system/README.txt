Student Management System (PHP OOP)

Files:
- index.php              Main page
- formHandler.php        Handles form submission and validation
- displayData.php        Displays saved student data
- student.php            Student class
- studentRepository.php  File storage handler
- data.txt               Data file

How to run:
1. Put the folder in your web server directory.
2. Open index.php in your browser.
3. Add student data from the form.

Notes:
- Data is stored in data.txt
- Phone accepts 11 to 14 digits and optional +
