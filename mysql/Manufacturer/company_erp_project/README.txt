Company Mini ERP Project
========================

Features:
1. Add manufacturer
2. Add product with stock
3. Sell product and create invoice
4. Print invoice
5. Return product from invoice
6. Automatic stock decrease on sale
7. Automatic stock increase on return

Files:
- setup.sql
- db.php
- helpers.php
- header.php
- footer.php
- style.css
- index.php
- manufacturer.php
- product.php
- sell.php
- invoices.php
- invoice.php
- return.php

How to Run:
1. Copy the folder to htdocs.
2. Open phpMyAdmin.
3. Create/import by running setup.sql.
4. Open: http://localhost/company_erp_project/index.php

Important:
- Database name: company
- Default XAMPP user: root
- Password: blank
- For invoice print, open an invoice and click Print Invoice.
