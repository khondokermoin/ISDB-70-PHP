POS Project Setup
=================

1. Copy the "pos_project" folder into htdocs (XAMPP) or www (WAMP).
2. Open phpMyAdmin.
3. Create/import the database by running setup.sql.
4. Open in browser:
   http://localhost/pos_project/add_manufacturer.php

Files included:
- setup.sql
- db_connection.php
- navbar.php
- add_manufacturer.php
- view_products.php
- README.txt
