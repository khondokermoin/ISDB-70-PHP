Company Project Setup
=====================

Files:
1. index.php
2. setup.sql

How to run:
1. Copy the folder "company_project" into htdocs.
2. Open phpMyAdmin.
3. Create/import by running setup.sql.
4. Open in browser:
   http://localhost/company_project/index.php

This project includes:
- Manufacturer add using stored procedure
- Product add using stored procedure
- Manufacturer delete using trigger
- Product view using SQL view

Database name:
company
